# Animation Skills For This Project

Use these skills whenever the user's request touches animation, motion design, WebGL, Three.js, SVG animation, interaction polish, route transitions, loading sequences, scroll effects, or high-aesthetic UI behavior.

## Core Motion Skills

- `emilkowal-animations`  
  Web UI motion feel, easing, duration, interaction timing, gestures, and animation polish.

- `gsap`  
  GSAP general reference for complex timelines, tweens, stagger, labels, nesting, playback, and performance.

- `gsap-core`  
  GSAP core API: `gsap.to()`, `from()`, `fromTo()`, easing, duration, stagger, defaults, responsive and reduced-motion patterns.

- `gsap-timeline`  
  Sequenced animation, timeline choreography, position parameters, nested timelines, and ordered transitions.

- `gsap-scrolltrigger`  
  Scroll-driven animation, pinning, scrub, parallax, reveal-on-scroll, and scroll-based triggers.

- `gsap-performance`  
  Performance guidance for smooth animation: transforms, opacity, batching, `will-change`, and avoiding layout thrashing.

- `gsap-plugins`  
  GSAP plugin usage: Flip, Draggable, ScrollTo, ScrollSmoother, SplitText, ScrambleText, SVG drawing, and plugin registration.

## Three.js / WebGL Motion Skills

- `threejs-animation`  
  Three.js keyframes, procedural motion, animation mixers, morph targets, skeletal animation, and object animation.

- `threejs-interaction`  
  Three.js input handling, raycasting, pointer/touch interactions, selection, and interactive 3D scenes.

- `threejs-shaders`  
  GLSL shaders, uniforms, custom visual effects, fragment/vertex animation, and ShaderMaterial work.

- `threejs-materials`  
  Material behavior, textures, PBR/basic/phong materials, and shader material styling.

- `threejs-geometry`  
  Geometry creation, BufferGeometry, custom vertices, instancing, and optimized mesh structures.

- `threejs-fundamentals`  
  Scene setup, cameras, renderers, Object3D hierarchy, transforms, coordinate systems, and render loops.

- `webgpu-threejs-tsl`  
  WebGPU-enabled Three.js, TSL shaders, node materials, compute shaders, GPU-driven effects, and WGSL/TSL patterns.

## SVG / Graphic Motion Skills

- `svg-animations`  
  SVG path animation, morphing, masks, gradients, loading icons, animated logos, motion paths, and CSS/SMIL SVG effects.

- `SVG Logo Designer`  
  SVG logo, icon, and visual mark design. Pair with `svg-animations` when the logo/icon should animate.

## Support Skill

- `frontend-design`  
  High-quality frontend composition, visual direction, layout, typography, and interface polish. Use with motion skills for this personal blog's high-aesthetic interactive direction.

## Project Bias

The user's goal is a cool, highly animated, high-interaction personal blog. For relevant requests, actively consider:

- animated route transitions
- scroll choreography
- shader-driven atmosphere
- hover and pointer interactions
- SVG/logo motion
- loading and reveal sequences
- motion performance and reduced-motion fallbacks
