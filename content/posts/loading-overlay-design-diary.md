# Loading Overlay 设计日记

Loading 不只是等待提示，它是用户进入站点的第一秒。这个项目里的 loading 需要先覆盖所有内容，然后再把页面放出来。

## 时间线

- 静态 loading shell 先出现。
- Three.js overlay 接管。
- 切片下落露出页面。
- 全局 setting 和播放按钮再出现。

## 细节

如果 setting 先闪出来，整个体验就会显得没有被导演过。
