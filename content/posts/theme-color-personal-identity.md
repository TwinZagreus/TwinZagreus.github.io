# Theme color as personal identity

The theme switcher should feel like changing the site mood, not choosing a random palette.

Each base color needs enough derived tones for text, borders, surfaces, and motion accents.

## Constraint

If a component cannot be themed from the shared map, it is probably not ready to become part of the permanent interface.
